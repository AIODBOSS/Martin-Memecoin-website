# $MARTIN — The Original Pepe
Static site ready to deploy. Edit `index.html`, `assets/css/style.css`, `assets/js/main.js`, and replace SVGs in `assets/images/` with real artwork.

## Features
- Green, investor-friendly theme with glass + shadow polish.
- Compact, sticky header optimized for mobile.
- Hero banner with slow toggle (3.5s) fade.
- Martin Picture Collection — clickable sources, auto-fades every 2s.
- Community Gallery — fades every 2.2s.
- Copy-to-clipboard CA button.
- Fully responsive; small navbar on mobile (hamburger menu).

## Quick start
Open `index.html` in a browser, or deploy with any static host (Netlify, Vercel, GitHub Pages, Cloudflare Pages).

## Replace images
Drop your own images in `assets/images/` keeping the same file names, or update paths in `index.html`.
